# Sizing Guide

## How sizes are shown
Each product's available sizes and colors are listed on its product page as selectable variants. Only in-stock combinations can be added to the cart.

## Finding the right fit
Product descriptions note the fit (e.g. regular, slim, oversized) where relevant. If a specific product's measurements aren't listed on its page, ask and we'll look up the details.

## Between sizes
If you're between two sizes, sizing generally runs true to standard measurements unless the product description says otherwise (e.g. "runs small" or "oversized fit").

## Exchanging for a different size
See the Returns & Exchanges policy — a size exchange is handled as a return plus a new order for the correct size, subject to availability.
