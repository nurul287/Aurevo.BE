## Tech Stack

| Concern | Tool |
|---------|------|
| ORM | Drizzle ORM (`drizzle-orm` + `postgres`) |
| DB | Local Supabase Docker ‚Äî `postgresql://postgres:postgres@127.0.0.1:55322/postgres` |
| Auth | Supabase JWT (`SUPABASE_JWT_SECRET` + `jsonwebtoken`) |
| Storage | Browser ‚Üí BE (multer) ‚Üí Supabase Storage (service role key) |
| Validation | Zod on all request bodies + query params |
| Error handling | Global error handler + typed `AppError` subclasses |
| Rate limiting | `express-rate-limit` ‚Äî tiered per route group |
| Docs | `swagger-jsdoc` + `swagger-ui-express` at `GET /api/docs` |
| TypeScript | `strict: true` |

## Local-Only Rule

**NEVER touch production DB.** All changes go to local Supabase Docker only.
- Local DB: `postgresql://postgres:postgres@127.0.0.1:55322/postgres`
- Local API: `http://127.0.0.1:55321`
- Production URL (`SYNC_PROD_DATABASE_URL` in Aurevo.UI) is read-only reference, never written to.

To re-introspect: `$env:DATABASE_URL="postgresql://postgres:postgres@127.0.0.1:55322/postgres"; npx drizzle-kit introspect`

## Response Shape Standard

**List endpoint:**
```json
{
  "success": true,
  "data": [...],
  "meta": {
    "pagination": {
      "page": 1, "limit": 12, "total": 45,
      "totalPages": 4, "hasNext": true, "hasPrev": false
    }
  }
}
```

**Single / Created / Deleted:**
```json
{ "success": true, "data": { "id": "..." } }          // 200 or 201
{ "success": true, "message": "Deleted successfully" } // 200 DELETE
```

**Error:**
```json
{ "success": false, "error": { "code": "NOT_FOUND", "message": "...", "details": {} } }
```

## Error Codes

| Code | HTTP | When |
|------|------|------|
| `NOT_FOUND` | 404 | Resource doesn't exist |
| `UNAUTHORIZED` | 401 | No/invalid JWT |
| `FORBIDDEN` | 403 | Valid JWT, wrong role |
| `VALIDATION_ERROR` | 400 | Zod schema failed |
| `CONFLICT` | 409 | Duplicate slug, already cancelled |
| `BUSINESS_RULE` | 422 | Business rule violation |
| `RATE_LIMIT` | 429 | Too many requests |
| `STORAGE_ERROR` | 500 | Supabase Storage failed |
| `INTERNAL_ERROR` | 500 | Unhandled server error |

## AppError Classes

```ts
throw new NotFoundError("Product");              // 404 NOT_FOUND
throw new ValidationError("msg", details);       // 400 VALIDATION_ERROR
throw new UnauthorizedError();                   // 401 UNAUTHORIZED
throw new ForbiddenError("Admin required");      // 403 FORBIDDEN
throw new ConflictError("Slug already exists");  // 409 CONFLICT
throw new BusinessRuleError("Can't delete");     // 422 BUSINESS_RULE
```

## Zod Validation Pattern

```ts
// {domain}.schema.ts
export const createProductSchema = z.object({
  body: z.object({ name: z.string().min(1), slug: z.string().regex(/^[a-z0-9-]+$/) }),
});
export const getProductsSchema = z.object({
  query: z.object({
    page: z.coerce.number().int().positive().default(1),
    limit: z.coerce.number().int().min(1).max(100).default(12),
    search: z.string().optional(),
  }),
});

// In routes file
router.get("/", publicLimiter, validate(getProductsSchema), getProducts);
router.post("/", authenticate, requireAdmin, validate(createProductSchema), createProduct);
```

## Composite Filter Pattern (list endpoints)

```ts
const conditions: SQL[] = [eq(products.isActive, true)];
if (filters.categoryId) conditions.push(eq(products.categoryId, filters.categoryId));
if (filters.search) conditions.push(ilike(products.name, `%${filters.search}%`));

const [rows, [{ total }]] = await Promise.all([
  db.select().from(products).where(and(...conditions)).limit(limit).offset(offset),
  db.select({ total: count() }).from(products).where(and(...conditions)),
]);
```

## Controller Pattern

```ts
export const getProducts = async (req: Request, res: Response, next: NextFunction): Promise<void> => {
  try {
    const result = await ProductService.getProducts(req.query as ProductFilters);
    res.status(200).json({ success: true, ...result });
  } catch (err) { next(err); }
};
```

## Auth Middleware

```ts
import { authenticate, optionalAuth, requireAdmin } from "../../middlewares";

router.get("/me", authenticate, getProfile);          // must be logged in
router.get("/", optionalAuth, getCart);               // guest or auth
router.delete("/:id", authenticate, requireAdmin, deleteProduct); // admin only
```

Role is read from `app_metadata.role` in the Supabase JWT. Admin role: `"admin"` or `"service_role"`.

## REST URL Conventions

- Plural nouns: `/api/products`, `/api/categories`
- Lowercase hyphenated: `/api/product-images`
- Nested: `/api/products/:id/variants`, `/api/products/:id/images`
- Actions: `/api/orders/:id/cancel`, `/api/cart/migrate`

Standard GET list query params: `page`, `limit`, `search`, `sortBy`, `sortOrder`, plus domain-specific filters.
